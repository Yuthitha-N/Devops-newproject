DOCKER NODE JS EXPRESS APPLICATION – STEP BY STEP GUIDE

STEP 1: Install Node.js (If not installed)

Check node version:
node -v
npm -v

If not installed, install Node.js from official website.

STEP 2: Create Project Folder

mkdir docker-basic-app
cd docker-basic-app

STEP 3: Create package.json File

nano package.json

Paste:

{
"name": "docker-basic-app",
"version": "1.0.0",
"main": "app.js",
"scripts": {
"start": "node app.js"
},
"dependencies": {
"express": "^4.18.2"
}
}

Save and exit.

STEP 4: Install Dependencies

npm install

This creates node_modules folder.

STEP 5: Create app.js File

nano app.js

Paste:

const express = require('express');
const app = express();

app.get('/', (req, res) => {
res.send('Hello from Docker!');
});

app.listen(3000, () => {
console.log('App running on port 3000');
});

Save and exit.

STEP 6: Test Application Locally

node app.js

Open browser:
http://localhost:3000

You should see:
Hello from Docker!

Stop server:
Ctrl + C

STEP 7: Create Dockerfile

nano Dockerfile

Paste:

FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["node", "app.js"]

Save and exit.

STEP 8: Build Docker Image

Check Docker:
docker --version

Build image:
docker build -t docker-basic-app .

STEP 9: Run Docker Container

docker run -d -p 3000:3000 docker-basic-app

Explanation:
-d = Run in background
-p 3000:3000 = Map local port 3000 to container port 3000

STEP 10: Check Running Containers

docker ps

STEP 11: Access Application

Open browser:
http://localhost:3000

Output:
Hello from Docker!

STEP 12: Stop Container

docker ps
docker stop <container_id>

STEP 13: Remove Container (Optional)

docker rm <container_id>

STEP 14: Remove Image (Optional)

docker rmi docker-basic-app
